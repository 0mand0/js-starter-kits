Document Object Model DOM
dom و یا مدل شی گراء سند

این مدل در واقع یک استاندارد و یک API برای دسترسی ساده به اجزای سند های HTML و XML می باشد. به وسیله HTML DOM جاوااسکریپت توانایی دسترسی و تغییرات تمامی اجزای یک سند HTML را داراست. هنگامی که یک سند HTML یا به زبان ساده تر فایل شما که دارای پسوند .html است و یا هر فایلی که به یک فایل HTML ترجمه می شود و در browser (مرورگر اینترنت) نمایش داده می شود load بارگذاری شود، بلافاصله browser یک مدل DOM را برای page صفحه HTML شما تولید می کند، تا از طریق آن javascript توانایی دسترسی و تغییر تمامی اجزای صفحه را داشته باشد.

node

هر dom دارای چندین node می باشد، node ها با یک گره node ریشه که همان root node می باشد شروع شده و به حالت درختی دارای زیر مجموعه ای از node ها می باشند. گره node ریشه یا اولین node در یک سند document است. که بقیه node ها در زیر مجموعه document قرار می گیرند. در مدل درختی node ها، به node بالاتر والد و یا والدین گفته می شود، node والد دارای مجموعه ای node فرزندان است. بصورتی که document بالای تگ html قرار دارد، در یک مدل document تگ html بعنوان node ریشه می باشد. زیر مجموعه و یا فرزندان html دو تگ head و body می باشند.
